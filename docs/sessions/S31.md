## Work completed

### `nova/core/episode.py` — Tier 1 primitive

Immutable, content-addressed Episode dataclass per ADR-134.

- Frozen dataclass; mutation raises `FrozenInstanceError`
- SHA-256 hash over canonical JSON (sorted keys, no whitespace)
- Hash computed at construction via `__post_init__` + `object.__setattr__`
- `verify()` recomputes and compares — detects tampering even via
  the `object.__setattr__` back-door
- 70 lines, stdlib only

### Verification

Three properties confirmed via inline test:

- **Determinism** — identical inputs produce identical hashes
- **Immutability** — `FrozenInstanceError` on attribute assignment
- **Tamper detection** — forced back-door mutation caught by `verify()`

## Architectural significance

The primitive establishes the trust foundation for all higher tiers.
Content-addressing makes episodes self-identifying; canonical
serialization makes hashes depend on meaning, not formatting;
immutability is enforced at the type level, not by convention.

This sets the ceiling for what kind of memory Nova can have:
auditable, tamper-evident, append-only.
